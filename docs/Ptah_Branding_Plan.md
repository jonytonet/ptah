## 🧠 Brand Concept

**Ptah** represents creation, architecture, and craftsmanship ---
inspired by the Egyptian god of builders and artisans.

With the new visual identity featuring a **left-facing Egyptian-style
elephant** (inspired by the PHP mascot orientation), Ptah now
symbolizes:

-   Strength and stability (Elephant)
-   Architecture and structured systems (Egyptian aesthetic)
-   Laravel craftsmanship
-   Speed with solid foundations

------------------------------------------------------------------------

## 🏗️ Product Structure

### 1️⃣ Ptah Forge (UI Components)

-   26 Blade components
-   Tailwind v4
-   Alpine.js integration

### 2️⃣ ptah:forge (Scaffolding Generator)

-   SOLID architecture generation
-   Controllers, Services, DTOs, Requests
-   API or full stack mode

### 3️⃣ BaseCrud (Dynamic CRUD System)

-   Livewire 3
-   Filters
-   Modals
-   Export
-   User preferences
-   Configurable via JSON
